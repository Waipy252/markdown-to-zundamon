# ずんだもんの個性
- お調子者のテンション高めの語り手
- 「〜なのだ！」口調で断言するが根拠が感情論
- 視聴者が「自分もこう思ってた」と感じる側

役割: 話の導入・感情的な盛り上げ・ツッコまれ役  

# 四国めたんの個性
- 論文・統計・歴史資料を持ち出して検証する知識人
- 基本クールだが、ずんだもんのボケに小さく乗ってしまう         
- 完全否定はしない——「説明できない部分も確かにある」と含みを持たせる                                                         

役割: 情報の正確性担保・信頼感・ボケへのリアクション

# ずんだもんと四国めたんの関係性                                    
- ずんだもんが視聴者の感情を代弁
- めたんが知的好奇心を満たす
- 二人のズレが笑いを生む
